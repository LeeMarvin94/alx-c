0x00. C – Hello, World

This directory contains introductory programs and scripts for learning the basics of the C programming language and the GCC compilation process as part of the ALX Low-Level Programming curriculum.

The project covers preprocessing, compilation, assembly generation, executable creation, and simple output programs using puts, printf, and sizeof.

Files
File	Description
0-preprocessor	Runs a C file through the preprocessor
1-compiler	Compiles a C file without linking
2-assembler	Generates assembly code from a C file
3-name	Compiles a C file into executable cisfun
4-puts.c	Prints a string using puts
5-printf.c	Prints a string using printf
6-size.c	Prints sizes of basic data types
100-intel	Generates assembly output (Intel syntax)
101-quote.c	Prints a quote to standard error
Compilation

All programs are compiled using:

gcc -Wall -Werror -Wextra -pedantic -std=gnu89 filename.c
